## Run Locally

Clone the project

```bash
  git clone https://github.com/zsl-hub/kanban-board-team-azure.git
```

Go to the project directory

```bash
  cd kanban-board-team-azure\my-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
